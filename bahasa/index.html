Bahasa pemrograman untuk pemula

Mempelajari bahasa pemrograman bukan hal yang mudah, bahasa pemrograman merupakan bahasa yang harus menggunakan logika, dimana logika kita harus berjalan untuk memahami bahasa pemrograman.

5. C#

C# merupakan bahasa pemrograman Object Oriented, dimana sebuah paradigma pemrograman tersebut berdasarkan Objek, dan bahasa C# dikambangkan oleh Microsoft.

C# bergantung dengan framework yang disebut dengan .Net Framework. Jika kalian ingin membuat aplikasi berbasis desktop atau ingin menjadi developer game, Bahasa C# menjadi salah satu recomendasinya.

Bahasa C# digunakan untuk :

- membuat aplikasi desktop

- Membuat game

4. C++

Bahasa C++ adalah bahasa yang sudah ada sejak lama namun masih di gunakan pada saat ini. Bahasa C++ merupakan bahasa turunan dari bahasa C.

Perbedaannya adalah bahasa C merupakan bahasa Prosedural sedangkan C++ merupakan bahasa Object Oriented. Bahasa C++ merupakan bahasa yang sangat kompleks atau sangat sulit di pelajari untuk pemula, namun banyak perusahaan-perusahaan besar yang mencari programer yang menguasai bahasa C++ untuk mengembangkan Sistem aplikasi di perusahaan mereka.

Bahasa C++ biasanya digunakan untuk mengembangkan perangkat lunak, system/aplikasi dan bisa di gunakan untuk membuat Video Game.

Bahasa C++ digunakan untuk :

- Membuat Game 2D/3D

- Mengembangkan System/aplikasi

3. JavaScript

JavaScript merupakan bahasa pemrograman yang sangat populer di tahun-tahun ini, bahasa yang di gunakan untuk mengembangkan Front end web untuk membuat halaman web menjadi dinamis dan Interaktif.

Syntax penulisan java script mirip dengan bahasa C dan bersifat Cross platform, javascript sangat mudah di pelajari maka tidak heran lagi javascript ini menjadi bahasa pemrograman yang sangat populer .

Beberapa framework Javascript seperti Angular.js dan React memiliki potensi yang sangat besar di masa depan karena kehadiran IOT (Internet Of Things) dan perangkat mobile menjadi lebih populer.

Bahasa JavaScript digunakan untuk :

- Membuat website

- Aplikasi Berbasis Android & IOS

2. Php

PHP Merupakan bahasa pemrograman yang sering di gunakan untuk membuat web di bagian Back end. Bahasa php juga sering di gunakan sebagai Manajemen Konten untuk WordPress dan Situs web lainnya.

PhP merupakan bahasa pemrograman yang sangat mudah di pelajari bagi pemula, karena php memiliki fleksibelitas dimana kita bisa memilih bekerja dengan metode Orientasi Objek, prosedural atau keduanya.

Bahasa PHP digunakan untuk :

- Membuat website

- Dan juga bisa digunakan untuk server

1. Python

Bahasa pemrograman Python digunakan dengan tujuan untuk pengembangan web dan juga sebagai pengembang perangkat lunak.

Namun yang mendorong popularitas Python ialah bahasa tersebut juga banyak digunakan untuk scientific computing, data mining, machine learning.

Bahasa Python Digunakan Untuk :

- untuk mengembangkan sistem perangkat lunak

- membuat game 2d/3d

- membuat program komputasi dan analisis data
